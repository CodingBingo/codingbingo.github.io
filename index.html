25岁写给未来孩纸的一封信

发表于 2018-06-01 | 分类于 给未来的孩子 | | 阅读次数:

Hello，25岁，不早也不迟，如约我来了！
今天大概是我这些年来做的最重大的决定之一吧，我从阿里离职了，很感激这家公司曾经给我的一切，无论是从技术层面还是从人性的修炼上而言，都让我有很大程度上的提升。一路走来，还是很感激身边的所有的同事们，或许我不是很尊老，但是他们很爱幼，嘿嘿。孩子，也希望你在成长的道路上，也能像我这样，会遇到的人和事都很nice，愿你常怀一颗感恩之心。
好啦，回过头来看看这一阶段的我吧，但愿在你看来，这一阶段的我是没有被荒废的，这一阶段的我的每一个决定都是正确的。
13年下半年，也就是上一封信的同年，我申请从软件工程专业转到卓越学院，那会儿说实话，有过犹豫，有过徘徊，有时候心里也在想—“你觉得这个地方（软件工程学院）差，或许你去那个地方不见得会好。”现在的我已经记不清楚当初的自己到底还想了些啥了，后面就是申请了转学院，接下来就是面试啦，我记得当初面试的时候，有个老师问我从我入学以来，我一共写了多少代码，我说大概3万行吧，然后我就以最低的学分被录取了。我记得我的第一台电脑是从小兰姐家拿来的很老的一台sony的上网本，电池没有，键盘坏的，就是这样的一台电脑，我第一年靠着他写了3万行代码，后来缠着我妈妈花了3900块买了一台华硕的电脑，孩子，希望未来的你能找到自己的兴趣所在，并为之努力。
转到卓越学院后，感谢同学和老师们，给我“大开方便之门”，让我逃掉了很多我那时觉得无所谓的课程，但是现在看起来，那时的自己还是有点幼稚的。后来学院给我申请了一个专门的实验室，那应该是我大学呆的最久的一个地方吧，我几乎所有的闲暇时间都呆在实验室。然后边学习边做一些练手的项目，后来遇到僧老师，开始带我做一些外包项目，忽然发现，原来我是可以通过自己所学所能为家里分担一些负担的，从那以后我基本没从家里拿过钱了，我记得那个时候，第一次拿到1000的工资的时候，陪我妈妈逛了商场买了一些东西好像，觉得自己好像忽然长大了一样。孩子，我想说的不是希望未来的你的第一笔工资要给我买礼物啥的，只是希望你能勇敢的去做你喜欢的事情，总会有一点成绩的，或许这就叫天道酬勤吧。
后面的日子做了不少外包，也赚了不少钱，给自己换了手机、电脑，在姐姐出嫁的时候，我拿出来我身上仅剩的10000块钱，给她做嫁妆。想起来还是挺想哭的，以前每年都是一家四口人过年，忽然就剩下三个人过年了，冷清了好多，爸妈虽然说没啥，但是心里肯定和我一样，还是很多不舍的。孩子，现在的我不清楚未来的你会不会有兄弟姐妹，但是希望你能珍视你生命中的每一个人，人生就是这样，总有人要逐渐远去。
14年开年的时候，马上一大帮人毕业了，每天会有收破烂的大爷大妈在楼底，等待那些即将毕业的学子们把过去的课本当成废品卖给他们，那个时候我又突发奇想，去联系他们把那些收购的书本买回来（孩子，可能你会问我为啥不直接在学校里收，但是我要告诉你的是，学校某些部门不会让你这么干的，慢慢的你会懂的，这就是社会！），然后出售给那些需要他们的学弟学妹们。当时我记得我当初是以每本书一块钱左右的价格收过来的，大概4000本吧，那个时候我们叫了一辆卡车，装的满满一卡车，然后搬到学校的实验室里面，依稀记得那时的自己，累并快乐着。孩子，希望你也能为你自己想做的事情去努力，去累并快乐着。
14年的夏天，我和超哥买了辆“豪车”，我们花了2400块买了两辆山地车，准备从杭州骑行去厦门。一路上，我睡过路边、桥下、河边等等。我们花了12天左右从杭州骑行到厦门，全程2100公里左右，中过署，断过粮，缺过水，也在某个半山腰叫天天不应，叫地地不灵，只能硬着头皮透推这沉重的车和行李上山。路上遇到过邀请我们进去喝水吹风扇的小夫妻；遇到过一位大哥哥，陪我们骑了200公里，请我们吃饭住酒店；遇到一位温州那边的叔叔，陪我们一直骑行到厦门，一路上给我们分享他过往的人生经历。很感谢超哥一路上的扶持，感谢路上遇到的所有的人，感谢路上经过所有的美景。孩子，希望未来的你在人生的路途中，可能会有各种困难险阻，但希望你不轻言放弃，希望你会有个人像超哥那样，一直相互扶持，希望你的路上也会遇到那些愿意帮助你的人。
15年开年，大三下半年了，我的大学生涯也快步入结束了，回首过去的时间，每天上完课，然后就钻到实验室，晚上总是等到寝室要关门才回去。如果杭州的夜空能看到星星的话，我相信，每天都是繁星伴着我回去吧。这个时候，就要面对一个比较重大的决定，是读研还是去工作了。那时候其实我没有太大的犹豫，因为我确实不太喜欢每天坐在那，像高三那样，于是我就开始制作简历，投简历准备面试了。说到面试，还真是有个比较有趣的东西，就是一开始投阿里的时候，我投的是安全的，那时候的我看来，我希望去尝试一些不同的领域，于是乎，理所当然的挂了。后面我重新摆正了一下姿态，去投自己比较擅长的方向，客户端开发，经历过几轮面试之后就拿到offer了。那时候的我，就只投了阿里的简历，好像有种孤注一掷的感觉。
后面就是来支付宝这边实习了，当时还有个研究生学姐提前来支付宝实习了，世界真的很小，刚好那时候的我们刚好坐在对面。每天一大早，我们赶地铁、赶公交，从遥远的下沙，两个小时的路程，来到支付宝大楼。晚上再赶末班车回学校，在末班的地铁上睡着被乘务员叫醒：“嘿，小伙子，到站啦！到站啦！”。然后一路飞奔跑到寝室，有时候回来晚了，会爬寝室楼的水管爬上二楼走上去。也有时候把刚刚睡着的阿姨叫醒，那个时候，每个寝室楼总会有两个阿姨，一个好阿姨，一个“坏阿姨”，总是会“骂骂咧咧”的给我开门。然后黑灯瞎火的在寝室洗漱。你要问我累不累，那肯定累的，但是还是很开心和自豪的。所以孩子，希望你能找到那么一个地方或事情，他能给你你所有的期待。
实习快结束的时候，我记得那个时候，刚好学校选举公派去巴黎交流访学的项目，我记得我是在项目截至日期的时候才提交材料的，因为一开始的时候，我觉得我自己的能力可能不够标准，但是我们的辅导员何老师鼓励我去试试。我记得那个时候，上午在公司完成实习转正面试，下午跑回学校开始公派交流访学的面试，很幸运，被这个项目选上了。当时面试的时候，因为那边的老师可能比较需要一个偏向编程实践的学生，然后我就以最低学分被选上了。那段时间，我的小外甥女出生了，我真的很开心，从此我们家又多了一个小屁孩，赶在出国前夕，我买车票去看了一下糖糖，当我第一次抱着这个幼小的生命的时候，我似乎有些想哭，愿我们能守护你安静长大。孩子，或许在未来的某个时候，你也会来到我眼前，但愿那个时候我是有能力去守护你的，但愿这辈子，你不后悔跟我来到这个世界。
那年中秋节的时候，我们坐飞机飞往巴黎，那是我迄今为止这辈子离家最远的一次，从飞机的屏幕上看，当飞机从中国上空飞过的时候，我竟有些许想哭，那个时候我才知道，原来这就叫祖国。到了巴黎，你会发现，你与那边的一切是那么“格格不入”，早上去了就喝咖啡；地铁还能罢工；看病需要预约；还会有暴恐事件……在那边，我努力发挥自己的专长，努力完成实验室老师交给我的任务，搭建了一个给阿尔兹海默症病人使用的平台，在我做演完演示的ppt的时候，我知道，这一程，我交了一份满意的答卷。对啦，那个时候也还有一个比较重要的选择，就是要不要选择在那边读研究生，当时带我的那个导师给我说，如果我决定来读的话，会帮我申请奖学金，如你所知，我最后还是选择回国了，哎。孩子，我希望以后的你，不会畏惧选择，希望以后的我不会给你带来顾虑，希望以后的你的每一份决定，都会有我在背后支持着你。
回来了以后，大学也快结束了，也还差最后一个事情没有做完，那就是进行一次徒步。于是我准备好装备，从杭州坐车去昆明，然后开始徒步去往大理、丽江，一路上遇到了不少有趣的人，见过我不曾见过的美景，人生嘛，就应该有一次说走就走的旅行。
然后就被师兄“催着”来支付宝上班啦，来公司快两年了，感谢这个平台给了我对大公司所有的期待！说实话，要走的时候还是会有很多不舍的。兢兢业业的师兄、温婉的莫老师、帅气的远尘、程咬金等等，我很感激大家对我的照顾与帮助。但是这么长时间，我总有种缺憾，会觉得似乎缺少了些什么，觉得自己想去闯一闯，想去干点什么，让这个世界留下点我存在的痕迹，那些年吹过的牛逼，总是要去实现的！孩子，还是希望你，在未来的人生旅途里，翻越千山万水，归来仍是少年。

Java从头再来-垃圾收集

发表于 2018-04-04 | 分类于 Java | | 阅读次数:

都说Java的垃圾我们不需要管，但是垃圾收集之类的相关知识我们还是有必要了解一下的，做到知己知彼百战不殆。

回收算法简介

  • 标记清理算法

    这个算法相对而言是最近简单的，标记阶段标识出哪些内存可以被清理，在清理阶段将这些内存清理掉。算法的优点显而易见–快速高效，但是缺点也比较明显，容易产生内存碎片，导致后续分配大对象空间的时候没有连续空间，进而引发Full GC。

  • 复制算法

    把内存一份为二，每次用其中一块内存，每次内存清理的时候，将那块内存上还存活的对象复制到另外一块内存上，这样就不会出现内存碎片了，但是内存的利用率太低了。

  • 标记整理算法

    标记整理算法算是标记清理算法和复制算法的结合体，在标记清理完垃圾内存后，会对存留下来的对象做一次整理，这样就能避免产生内存碎片的问题。

转眼间，从我呱呱坠地到现在都有20年了，我也不知道是该感叹是时间过得太快还是自己还没成长起来……总之现在的我还是一无所有，除了那些所谓的青春。谁也不知道，再过几年，在社会的“摧残”下，我的青春还是否会剩下什么？当然，孩子，我这不是对社会的愤慨，只是觉得现在的自己在社会面前好渺小……
恍恍惚惚的走过了20年，我也不知道当你出生的时候，我为你创造的家会是怎么样，是贫穷还是富有？现在的我还是无法预知的，至少在我看来，我爸爸没有为我留下什么（当然这也不是抱怨啊！）。如果到你出生的时候我仍是一个幸苦的程序员，我允许你骂我一声无能，然后自己努力去学习去拼搏吧……当然，如果那个时候我给你的家境还算是不错的话，我还是希望你每年的夏天都回到我的故乡，那个我生活过19年的地方，我相信那里，你会学到很多很多，那里的山山水水也是很漂亮的，虽然平淡无奇，但对于城市出生的孩子而言，那一切都会是新鲜的！就像我刚刚来到城里面一样，公交，高楼大厦，电梯，KFC……等等，或许这些东西我是到大学才体会到的，当然，我可以保证让你在我之前体会到这些！
来看看我这些年吧，也算是让你看看，少走些弯路吧……，从我到初中的时候开始吧。在一开始的“家庭会议”讨论中是决定送我去县城的一家初中读书的，可是最后一次考试貌似我的成绩有点不堪，以致后来我就就近去了一所家旁边的初中，这样的话，依旧看见了我的“初恋”，不要笑话我啊，其实在我看来，那时候的我都不知道那是什么感觉，真的！但是就仅仅觉得那种感觉还是很让人舒服的，当然，戏剧性的是我的成绩比她还要差，这样也就成了我努力学习的动力，不要觉得匪夷所思，就像我高中时候的班主任说的那样——暗恋对人是有促进作用的。（其实当初他说的时候，我就很想说，这句话是对的！）依稀记得那个时候自己永远是寝室最后一个睡最先一个起来的，我也很佩服自己的这份坚持与努力，孩子，希望以后的你也会像我当初那样努力（当然我并不是说你应该去暗恋一个比你成绩好的女生）。
再到后来的高中吧，看到我高中母校开出的“美好”政策，我还是决定去了，本来家里就不是那般富裕，加上暗恋的她也去了那，没理由不去啊！高中在我脑海里，现在想起来，除了高一的时候还算舒服吧，到后来的感觉吧，都是一个累，让我在那个时候都想下定决心不让你去读高中，在家我教了得了（当然这只是我想想而已啊，在学校和同龄人还是能学到不少的东西的！）。当然，后来我们就分配到了一个班级了，接下来一切都好像是水到渠成了是吧？？？可是后来有很多原因就……（当然我觉得自己是错的很多的啊！），哭哭啼啼的就跑回家了，抱着爸妈痛哭！我想说，感谢爸妈那个时候没有打骂我，一直在我身边！孩子，虽然我不知道自己在未来的时间能为你留下什么，但是我可以很肯定的说无论你干什么，我永远都是会支持你的！至于后来的高中就只剩下自己忙碌的身影了，其实在我看来，那时的忙碌大部分是为了心里不再空虚罢了！至少现在的我一闲下来就觉得很是空虚！后来到了暑假了，那个暑假也是我最长的暑假了，我第一次出我们伟大的安徽省，爸妈让我去找点事情干干，但是毕竟还是个孩子嘛，确实干不下去，玩了两个月啊！只有一个字爽……
接下来就是大学了，感觉吧，高中自己的努力还是没有白费的，来到了杭州这座城市，也终于圆了我出省的那个想法，可是当正真出了省之后你会发现，当初你的那些简单的想法或许是错的哦……就像以前的一句话，城里的人想出来，城外的人想进去。其实当你在这个陌生的城市，看着那些同学一些小假都能回家，你会觉得自己是很孤单的！
孩子，希望你在以后做出的每一次决定的时候都能很慎重的想好，因为你的将来需要对你现在的决定负责，但是一旦决定了就不要再后悔了，因为那一点没有用处！……大学了，人好像一夜之间突然就长大了，其实我是很觉得奇妙的，明明上半年谈恋爱就是早恋，可是到了下半年，你再谈恋爱就没有什么了。其实我是很想问下这个原因是什么，或许这个社会就这样吧！所以，孩子，我不反对你喜欢一个人，也不反对你在任意的时间谈恋爱，只希望你能不要后悔，要明白自己在干什么，要安排好自己的时间……就如我以前想的一样，我对电脑还真不是一般的热爱，我花了半年的时间学完了别人一年需要学习的东西，说实话，那段日子，每天呆在机房里面，编程是我唯一的事情！说实话那段时间过得简单幸福，虽然很累。……再说感情方面吧，大学，来自全国各地的女生！诚然比高中那时代要多得多，诚然也遇到过几个让我心动的女孩子，或许在我看来吧，我自己都不是很清楚自己到底是不是喜欢了！（补充一句吧，你喜欢的人不一定喜欢你，就像喜欢你的人你也不一定喜欢一样，也许在某段时间，你会发现你不喜欢以前非常喜欢的人了，但是当真正分开之后，你会发现真正喜欢的确实原来的那个，所以千万不要三心二意的！感情上没什么对错，相信生活是好的就好！）所以孩子，初恋还是不要太早了吧，在我看来，那真的很难忘掉，就像我一样，会在无意的无聊中写下她的名字，不过话说回来，写的蛮好看的！还有其他的事情就不过多说了，以后要是有机会再说吧，但是我还想说一句，大学努力学真能学到很多的东西的！
其实在我看来，我是不明白大部分人的迷茫的，他们不知道将来的他们能干什么，不知道将来的他们会干什么，或许连他们自己都不知道问什么他们会来到软件工程专业吧……，不过，孩子，希望在我这个时候的你是清晰的，知道自己想要的是什么，知道自己想干什么……那样会是极好的！
好了，看来也该是时候结束了，本来写的时候，心里好像有说不完的话，但是写起来的时候发现自己写不了那么多的……好了，期待在我25岁时候写给你的是什么了，当然我也期待在我25岁的时候我还能写给你什么样的话了，希望你在看我20岁时候留给你的东西的时候，你能学到一些什么东西！希望你是开心的快乐的……
