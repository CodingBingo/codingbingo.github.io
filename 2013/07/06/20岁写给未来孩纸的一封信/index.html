20岁写给未来孩纸的一封信

发表于 2013-07-06 | 分类于 给未来的孩子 | | 阅读次数:

转眼间，从我呱呱坠地到现在都有20年了，我也不知道是该感叹是时间过得太快还是自己还没成长起来……总之现在的我还是一无所有，除了那些所谓的青春。谁也不知道，再过几年，在社会的“摧残”下，我的青春还是否会剩下什么？当然，孩子，我这不是对社会的愤慨，只是觉得现在的自己在社会面前好渺小……

恍恍惚惚的走过了20年，我也不知道当你出生的时候，我为你创造的家会是怎么样，是贫穷还是富有？现在的我还是无法预知的，至少在我看来，我爸爸没有为我留下什么（当然这也不是抱怨啊！）。如果到你出生的时候我仍是一个幸苦的程序员，我允许你骂我一声无能，然后自己努力去学习去拼搏吧……当然，如果那个时候我给你的家境还算是不错的话，我还是希望你每年的夏天都回到我的故乡，那个我生活过19年的地方，我相信那里，你会学到很多很多，那里的山山水水也是很漂亮的，虽然平淡无奇，但对于城市出生的孩子而言，那一切都会是新鲜的！就像我刚刚来到城里面一样，公交，高楼大厦，电梯，KFC……等等，或许这些东西我是到大学才体会到的，当然，我可以保证让你在我之前体会到这些！
来看看我这些年吧，也算是让你看看，少走些弯路吧……，从我到初中的时候开始吧。在一开始的“家庭会议”讨论中是决定送我去县城的一家初中读书的，可是最后一次考试貌似我的成绩有点不堪，以致后来我就就近去了一所家旁边的初中，这样的话，依旧看见了我的“初恋”，不要笑话我啊，其实在我看来，那时候的我都不知道那是什么感觉，真的！但是就仅仅觉得那种感觉还是很让人舒服的，当然，戏剧性的是我的成绩比她还要差，这样也就成了我努力学习的动力，不要觉得匪夷所思，就像我高中时候的班主任说的那样——暗恋对人是有促进作用的。（其实当初他说的时候，我就很想说，这句话是对的！）依稀记得那个时候自己永远是寝室最后一个睡最先一个起来的，我也很佩服自己的这份坚持与努力，孩子，希望以后的你也会像我当初那样努力（当然我并不是说你应该去暗恋一个比你成绩好的女生）。
再到后来的高中吧，看到我高中母校开出的“美好”政策，我还是决定去了，本来家里就不是那般富裕，加上暗恋的她也去了那，没理由不去啊！高中在我脑海里，现在想起来，除了高一的时候还算舒服吧，到后来的感觉吧，都是一个累，让我在那个时候都想下定决心不让你去读高中，在家我教了得了（当然这只是我想想而已啊，在学校和同龄人还是能学到不少的东西的！）。当然，后来我们就分配到了一个班级了，接下来一切都好像是水到渠成了是吧？？？可是后来有很多原因就……（当然我觉得自己是错的很多的啊！），哭哭啼啼的就跑回家了，抱着爸妈痛哭！我想说，感谢爸妈那个时候没有打骂我，一直在我身边！孩子，虽然我不知道自己在未来的时间能为你留下什么，但是我可以很肯定的说无论你干什么，我永远都是会支持你的！至于后来的高中就只剩下自己忙碌的身影了，其实在我看来，那时的忙碌大部分是为了心里不再空虚罢了！至少现在的我一闲下来就觉得很是空虚！后来到了暑假了，那个暑假也是我最长的暑假了，我第一次出我们伟大的安徽省，爸妈让我去找点事情干干，但是毕竟还是个孩子嘛，确实干不下去，玩了两个月啊！只有一个字爽……
接下来就是大学了，感觉吧，高中自己的努力还是没有白费的，来到了杭州这座城市，也终于圆了我出省的那个想法，可是当正真出了省之后你会发现，当初你的那些简单的想法或许是错的哦……就像以前的一句话，城里的人想出来，城外的人想进去。其实当你在这个陌生的城市，看着那些同学一些小假都能回家，你会觉得自己是很孤单的！
孩子，希望你在以后做出的每一次决定的时候都能很慎重的想好，因为你的将来需要对你现在的决定负责，但是一旦决定了就不要再后悔了，因为那一点没有用处！……大学了，人好像一夜之间突然就长大了，其实我是很觉得奇妙的，明明上半年谈恋爱就是早恋，可是到了下半年，你再谈恋爱就没有什么了。其实我是很想问下这个原因是什么，或许这个社会就这样吧！所以，孩子，我不反对你喜欢一个人，也不反对你在任意的时间谈恋爱，只希望你能不要后悔，要明白自己在干什么，要安排好自己的时间……就如我以前想的一样，我对电脑还真不是一般的热爱，我花了半年的时间学完了别人一年需要学习的东西，说实话，那段日子，每天呆在机房里面，编程是我唯一的事情！说实话那段时间过得简单幸福，虽然很累。……再说感情方面吧，大学，来自全国各地的女生！诚然比高中那时代要多得多，诚然也遇到过几个让我心动的女孩子，或许在我看来吧，我自己都不是很清楚自己到底是不是喜欢了！（补充一句吧，你喜欢的人不一定喜欢你，就像喜欢你的人你也不一定喜欢一样，也许在某段时间，你会发现你不喜欢以前非常喜欢的人了，但是当真正分开之后，你会发现真正喜欢的确实原来的那个，所以千万不要三心二意的！感情上没什么对错，相信生活是好的就好！）所以孩子，初恋还是不要太早了吧，在我看来，那真的很难忘掉，就像我一样，会在无意的无聊中写下她的名字，不过话说回来，写的蛮好看的！还有其他的事情就不过多说了，以后要是有机会再说吧，但是我还想说一句，大学努力学真能学到很多的东西的！
其实在我看来，我是不明白大部分人的迷茫的，他们不知道将来的他们能干什么，不知道将来的他们会干什么，或许连他们自己都不知道问什么他们会来到软件工程专业吧……，不过，孩子，希望在我这个时候的你是清晰的，知道自己想要的是什么，知道自己想干什么……那样会是极好的！
好了，看来也该是时候结束了，本来写的时候，心里好像有说不完的话，但是写起来的时候发现自己写不了那么多的……好了，期待在我25岁时候写给你的是什么了，当然我也期待在我25岁的时候我还能写给你什么样的话了，希望你在看我20岁时候留给你的东西的时候，你能学到一些什么东西！希望你是开心的快乐的……
